# GTAVC-VehicleSpawner
Vehicle Spawner for GTA Vice City

----------- Installation
First you have to install CLEO. You can download it here. Copy (.cs) file in the package to CLEO folder.

----------- How To Use
On Foot:
Press F5 to spawn vehicle. Press again to change.

This mod will spawn 4 vehicles: White Admiral, White PCJ 600, White Cheetah and White Maverick.
